# Tool: Microsoft 365

## Problem 1: Cannot sign in to Microsoft 365
**Solution:**
This usually happens due to incorrect credentials or account lock.
Steps:
1. Re-enter username and password carefully
2. Check Caps Lock and keyboard language
3. Try signing in from browser
4. Contact IT if account is locked

---

## Problem 2: Microsoft 365 apps not opening
**Solution:**
This occurs due to license or installation issues.
Steps:
1. Restart the computer
2. Check Microsoft 365 license status
3. Repair Microsoft 365 from Control Panel
4. Update the application

---

## Problem 3: License not activated
**Solution:**
License activation may fail due to account or network issues.
Steps:
1. Sign out and sign in again
2. Check assigned license in Microsoft portal
3. Ensure internet connectivity
4. Restart Microsoft app

---

## Problem 4: Teams not working properly
**Solution:**
Teams issues often occur due to cache or network problems.
Steps:
1. Close Teams completely
2. Clear Teams cache
3. Restart Teams
4. Check internet connection

---

## Problem 5: OneDrive not syncing files
**Solution:**
Sync issues occur due to storage or sign-in problems.
Steps:
1. Check OneDrive storage quota
2. Restart OneDrive
3. Sign out and sign in again
4. Restart computer

---

## Problem 6: Outlook not syncing in Microsoft 365

4. Restart system

---

## Problem 7: Cannot access SharePoint site
**Solution:**
Permission or login issues usually cause this.

1. Confirm access permission with IT

---

## Problem 8: Files missing in OneDrive
**Solution:**
Files may be moved, deleted, or not synced.
Steps:
1. Check OneDrive recycle bin
2. Search for file name
3. Check sync status
4. Restore file if needed

---

**Solution:**
Corrupted add-ins or templates may cause crashes.
Steps:
1. Open Word in Safe Mode
2. Disable add-ins
3. Repair Microsoft 365
4. Update Word

---

## Problem 14: Cannot install Microsoft 365
**Solution:**
Installation may fail due to permissions or antivirus.

1. Run installer as administrator
2. Disable antivirus temporarily
3. Ensure enough disk space
4. Restart installation

---
